# Corporate-Landing-Page
A Fully Responsive Corporate Landing Page using HTML and CSS


The page  includes:


* Hero section


* Features grid


* Testimonials carousel (CSS-only implementation)


* Pricing cards


* Footer


The design is fully responsive and work seamlessly across:


* Mobile


* Tablet


* Desktop


Ensures clean structure, semantic HTML, maintainable CSS, and proper layout techniques (Flexbox/Grid).
